# HealthForge Mermaid Diagrams

This folder contains architecture and data-flow Mermaid diagrams.

- `01-system-flow.mmd` - End-to-end app flow (UI, API, auth, AI, queue, WhatsApp).
- `02-data-model-er.mmd` - Core SQLite schema and relationships.
- `03-notification-status-state.mmd` - Notification lifecycle states.
- `04-file-to-table-map.mmd` - Which files/routes read/write which DB tables.

You can paste each `.mmd` file into Mermaid-enabled Markdown, or render them in tools that support Mermaid.
